# ClearBag dev環境 手動E2Eテストチェックリスト

**作成日**: 2026-03-03
**対象URL**: https://clearbag-dev.firebaseapp.com/
**目的**: デプロイ後の基本動作を手動で確認するためのチェックリスト。今後テストケースを追加していく前提で運用する。

---

## 実行方法

1. 以下のチェックリストを上から順に実行する
2. 各項目の結果を `OK` / `NG` / `SKIP` で記録する
3. NG の場合は備考欄に再現手順・スクリーンショット等を添える
4. 新しいテストケースは該当セクションの末尾に追番で追加する
